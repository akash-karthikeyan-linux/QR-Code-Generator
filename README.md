## âš™ï¸ Installation

To use the script, you first need to install the `qrcode` library along with the `Pillow` (PIL) dependency for image generation.

1.  Open your IDE's **terminal** or your system's command prompt.
2.  Execute the following command:

    ```bash
    pip install qrcode[pil]
    ```

    > This command installs the necessary Python packages to generate and save the QR code image file.
    
3.  After installation copy the code from qr_generator.py and execute it.
